# iFixit Repair Telegram Bot
<h1 align="center">
   Telegram бот для поиска инструкций по ремонту техники 

Основные возможности

     Поиск по модели устройства (/repair iPhone 12)
     Иерархический навигатор: от категории → модели → конкретной поломки
     Показ шагов ремонта с картинками и текстом
     Прямые ссылки на оригинальные руководства iFixit
     История поиска (/history) — смотри, что уже искал
     Поддержка русского языка — приоритетно ищет русские версии руководств
